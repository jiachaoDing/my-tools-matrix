Productivity FAQ

Will it work if I switch tabs?
Yes. The calculation logic uses the difference between the starting system time and the current system time. Even if the browser throttles the background tab, the time will sync perfectly the moment you return.
What are the Pomodoro presets?
We include common productivity blocks like 25 minutes (standard focus block). This allows you to quickly start deep-work sessions without manually typing in the duration every time.
