Symbol FAQ

Will these symbols work in Word/Google Docs?
Yes. These are standard Unicode characters. When you copy them from here and paste them into any modern text editor or document processor, they will maintain their appearance regardless of the font.
What are HTML Entities?
HTML Entities (like &copy;) are used by web developers to display special characters in HTML code without causing issues with the code's structure. Our tool provides these codes for easier development.
