FAQ

What counts as a special character?

Special characters include symbols like @, #, $, %, &, *, etc., as well as non-alphanumeric characters that aren't letters, numbers, spaces, or standard punctuation.

How are letters counted?

Letters include all alphabetic characters (A-Z, a-z) in any language. The character breakdown shows which specific letters appear in your text.

What about non-English characters?

The tool works with any Unicode characters, including accented letters, non-Latin scripts, and emojis.

Is my text secure?

Absolutely. All text processing happens locally in your browser. Your content never gets uploaded to any server.

Can I analyze code or programming text?

Yes, the character counter works perfectly with code, markup, and any other text content, accurately counting all characters and symbols.
