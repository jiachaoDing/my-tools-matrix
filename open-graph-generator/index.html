Open Graph FAQ

                    

                        

                            
                                What is the ideal OG image size?
                                
                                    
                                
                            
                            

                                The recommended size is 1200 x 630 pixels for the best quality across Facebook, LinkedIn, and Discord. The minimum size is 600 x 315 pixels.
                            

                        

                        

                            
                                Which platforms use Open Graph?
                                
                                    
                                
                            
                            

                                Facebook, LinkedIn, Discord, Slack, and WhatsApp primarily use Open Graph. Twitter also supports OG tags if its own Twitter Card tags are missing.
