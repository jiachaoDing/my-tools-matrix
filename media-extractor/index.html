Frequently Asked Questions

What file extensions are supported?
We support common image formats (JPG, PNG, GIF, WebP, SVG) and video formats (MP4, WebM, MOV).
Does this tool download files?
Due to browser security restrictions, the "Download" button opens the media in a new tab where you can right-click and "Save As". This ensures the process remains 100% private and client-side.
Why use a Base URL?
Many websites use relative paths (like "/images/logo.png") in their code. By providing the base domain, our tool can reconstruct the full URL so you can view and download the media.
