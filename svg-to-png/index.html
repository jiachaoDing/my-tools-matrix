SVG to PNG FAQ

Will my SVG lose quality during conversion?
No. Because SVG is a vector format, it can be scaled to any size before being rasterized into a PNG. We ensure the resulting PNG is sharp and clear.
How does scaling work?
The scale factor multiplies the original dimensions defined in the SVG file. For example, if an SVG is 32x32px and you choose 4x scaling, the resulting PNG will be 128x128px.
Is it possible to convert PNG back to SVG?
No, converting a raster image (PNG) back to a vector format (SVG) is a complex process called tracing. This tool only handles SVG to PNG conversion.
