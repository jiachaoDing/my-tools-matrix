Expert Q&A

क्या यह \n या \t जैसे escape sequences support करता है?
हाँ! जब "Use Regex" enabled होता है, तो आप सभी standard regular expression escape sequences इस्तेमाल कर सकते हैं, जिसमें newlines (\n), tabs (\t), और more complex patterns शामिल हैं।
"Whole Word Match" क्या है?
अगर आप "cat" को whole word match के साथ search करते हैं, तो यह "caterpillar" या "concatenate" को ignore कर देगा। यह सिर्फ spaces या punctuation से घिरे exact word "cat" को ही replace करता है।
मैं एक साथ कितनी files process कर सकता हूं?
Technically, files की संख्या पर कोई hard limit नहीं है। Tool को memory manage करने के लिए files को sequentially process करने के लिए optimized किया गया है। Large individual files (50MB से ज्यादा) processing के दौरान आपके browser को थोड़ा slow कर सकती हैं।
