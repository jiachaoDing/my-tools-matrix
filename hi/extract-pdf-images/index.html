Frequently Asked Questions

कौन से प्रकार की images extract की जा सकती हैं?
हमारा tool PDFs में embedded raster images को extract करता है, जिसमें JPEG, PNG, और अन्य bitmap formats शामिल हैं। इसमें photos, scanned images, और raster graphics शामिल हैं। Vector graphics (जैसे lines से बने charts या icons) images के रूप में extractable नहीं हैं।
मेरी PDF में कोई images क्यों नहीं मिलीं?
कुछ PDFs में सिर्फ text या vector graphics होते हैं जिन्हें images के रूप में extract नहीं किया जा सकता। इसके अलावा, scanned documents से बनाई गई PDFs content को अलग तरीके से store करती हैं। अगर आपकी PDF में images दिखती हैं लेकिन कोई नहीं मिलीं, तो visuals vector-based illustrations हो सकते हैं।
क्या कोई file size limit है?
चूंकि सभी processing आपके browser में locally होती है, limit आपके device's memory पर निर्भर करती है। ज्यादातर modern devices 100MB या उससे ज्यादा PDFs को बिना किसी issue के handle कर सकते हैं। बहुत बड़ी files के लिए processing ज्यादा समय ले सकता है।
Extracted images किस format में हैं?
सभी extracted images PNG files के रूप में save होती हैं ताकि maximum compatibility और quality preservation सुनिश्चित हो। PNG एक lossless format है जो image quality को maintain करता है चाहे आप कितनी बार save या edit करें।
