Frequently Asked Questions

Can I unlock a PDF without knowing the password?
No. This tool requires you to know the correct password. We do not bypass, crack, or break PDF encryption. This tool is designed for legitimate use cases where you know the password but want to create an unprotected copy.
Is my password and PDF secure?
Absolutely. Your PDF file and password never leave your device. All processing happens entirely in your browser using JavaScript. We don't have any servers that handle your documents or passwords.
What types of PDF protection can be removed?
This tool can remove user password protection (the password required to open the PDF). Once you provide the correct password, we create a new PDF without any password requirements.
How do I add password protection back?
You can use our Protect PDF tool to add password protection to any PDF document, including ones you've previously unlocked.
