Frequently Asked Questions

Why does the art look broken when I paste it?
ASCII art requires a **monospaced font** (fixed-width) to align correctly. If you paste it into an editor or site using proportional fonts, the characters won't line up. Ensure the receiving end is using a font like Courier or Consolas.
Can I use this for non-English text?
Figlet fonts are traditionally designed for ASCII character ranges. Some fonts might support special characters, but for languages like Chinese or Arabic, the standard Figlet fonts will likely not work.
How do I download the art?
You can simply copy the text using the "Copy ASCII Art" button. For batch processing, use the "Download All" button to save each art block as a separate .txt file.
