Markdown FAQ

What Markdown flavors are supported?
We support standard GFM (GitHub Flavored Markdown), which includes tables, task lists, and strikethroughs alongside standard Markdown syntax.
Can I batch process multiple files?
Yes! Simply drag multiple `.md` files into the input area or use the "Import Files" button. Each file will be converted and made available for download.
